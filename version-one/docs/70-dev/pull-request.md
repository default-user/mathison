# Pull Request Template

**Version:** 1.0.0
**Last Updated:** 2026-01-03

---

## Who This Is For

- Developers creating pull requests
- Reviewers evaluating contributions
- Maintainers tracking changes

---

## Template

```markdown
## Summary

[Brief description of changes]

## Type of Change

- [ ] Bug fix
- [ ] New feature
- [ ] Breaking change
- [ ] Documentation update

## Changes Made

- [List specific changes]

## Testing

- [ ] All tests pass (`pnpm -r test`)
- [ ] New tests added for new features
- [ ] Manual testing completed

## Security Considerations

- [ ] No new security vulnerabilities
- [ ] Credentials not committed
- [ ] AI-assisted code reviewed

## Contributor Certificate

I certify that this contribution complies with the Contributor Certificate.

Signed: [Name] <email>
Date: YYYY-MM-DD
```
